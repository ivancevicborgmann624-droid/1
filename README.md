
# 小马宝莉角色与剧情介绍网站 🌈

这是一个关于《小马宝莉：友谊就是魔法》的粉丝网站，包含主要角色介绍、剧情概览和精彩剧集推荐。

## 📁 项目结构

```
mlp/
├── index.html      # 主页面文件
├── styles.css      # 样式文件
└── README.md       # 项目说明文档
```

## 🚀 如何使用

### 方法1：直接打开

1. 在文件浏览器中找到 `mlp` 文件夹
2. 双击 `index.html` 文件
3. 网页将在默认浏览器中打开

### 方法2：使用本地服务器

如果你安装了Python，可以使用以下方法：

```bash
# 进入项目目录
cd e:\pandas-practicei_study\mlp

# 启动本地服务器
python -m http.server 8000
```

然后在浏览器中访问：`http://localhost:8000`

## 💡 自定义修改

### 修改颜色主题
在 `styles.css` 中修改以下颜色值：
```css
/* 主色调 */
--primary-color: #667eea;
--secondary-color: #764ba2;
```

### 添加新角色
在 `index.html` 的角色部分添加新的 `character-card` 块

### 添加新剧集
在 `index.html` 的剧集部分添加新的 `episode-card` 块
